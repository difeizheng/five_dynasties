# 📤 百度网盘自动上传使用说明

*使用 Playwright 自动化上传文件到百度网盘*

---

## 📋 准备工作

### 1. 已安装的工具

✅ Node.js（已安装）
✅ Playwright（已安装）
✅ Chromium 浏览器（已安装）

### 2. 文件位置

| 文件 | 路径 |
|------|------|
| **上传脚本** | `D:\openclaw\workspace\history\baidu_upload.py` |
| **要上传的文件** | `D:\openclaw\workspace\history\` 下的所有文件 |
| **Cookie 保存位置** | `D:\openclaw\workspace\history\baidu_cookie.json` |

---

## 🚀 使用步骤

### 第一次使用（需要手动登录）

#### 步骤 1：运行脚本

打开 PowerShell 或命令提示符，运行：

```powershell
cd D:\openclaw\workspace\history
python baidu_upload.py
```

或者直接双击运行 `baidu_upload.py`

#### 步骤 2：手动扫码登录

脚本运行后会：
1. 自动打开 Chromium 浏览器
2. 打开百度网盘首页
3. 显示登录二维码
4. **你需要用手机百度 APP 扫码登录**

⏰ **登录时间：60 秒**

#### 步骤 3：保存登录状态

登录成功后：
- 脚本会自动保存 Cookie 到 `baidu_cookie.json`
- 下次运行就不需要再扫码了！

#### 步骤 4：自动上传

登录成功后，脚本会：
- 自动遍历要上传的文件
- 逐个上传到百度网盘
- 显示上传进度

---

### 第二次及以后（自动登录）

## 🔧 常见问题

### Q1: 提示 "playwright 未安装"

**解决：**
```powershell
npm install -g playwright
npx playwright install chromium
```

### Q2: 扫码登录超时

**解决：**
- 脚本会等待 60 秒
- 如果超时，重新运行脚本即可
- 确保手机百度 APP 已登录

### Q3: Cookie 失效

**解决：**
- 删除 `baidu_cookie.json`
- 重新运行脚本，重新扫码

### Q4: 上传失败

**可能原因：**
1. 网络问题 → 检查网络连接
2. 文件太大 → 百度网盘限制单文件 4GB（非会员）
3. 浏览器被拦截 → 关闭杀毒软件试试

### Q5: 找不到文件

**解决：**
- 检查文件路径是否正确
- 确保文件在 `D:\openclaw\workspace\history\` 文件夹中

## 💡 使用技巧

### 技巧 1：定期清理 Cookie

如果登录状态异常：
```powershell
del D:\openclaw\workspace\history\baidu_cookie.json
```
然后重新运行脚本

### 技巧 2：后台运行

可以在后台运行脚本：
```powershell
Start-Process python -ArgumentList "baidu_upload.py" -WindowStyle Hidden
```

### 技巧 3：定时上传

用 Windows 任务计划程序，定时运行脚本
