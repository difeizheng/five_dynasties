## 🏛️ 中央官职详解

### 一、最高行政长官（相当于国务院总理）

| 朝代 | 官职 | 品级 | 职责 | 著名人物 |
|------|------|------|------|---------|
| **秦** | 丞相 | 正一品 | 辅佐皇帝，总揽政务 | 李斯 |
| **汉** | 丞相/相国 | 正一品 | 同上 | 萧何、曹参 |
| **汉** | 大将军 | 正一品 | 军事最高长官 | 霍光、曹操 |
| **唐** | 同中书门下平章事 | 正一品 | 宰相，集体决策 | 房玄龄、杜如晦 |
| **唐** | 中书令 | 正一品 | 起草诏令 | 张说 |
| **唐** | 侍中 | 正一品 | 审核诏令 | 魏徵 |
| **宋** | 同平章事 | 正一品 | 宰相 | 王安石、司马光 |
| **宋** | 参知政事 | 从一品 | 副宰相 | 范仲淹 |
| **元** | 中书省丞相 | 正一品 | 宰相 | 耶律楚材 |
| **明** | 丞相（洪武后废） | 正一品 | 宰相 | 胡惟庸（最后一任） |
| **明** | 内阁大学士 | 正五品→正一品 | 实际宰相 | 张居正 |
| **清** | 内阁大学士 | 正一品 | 名义宰相 | 曾国藩 |
| **清** | 军机大臣 | 无品级（兼职） | 实际宰相 | 和珅、李鸿章 |

### 丞相权力演变

```
秦：丞相一人独大 → 汉：左右丞相分权 → 唐：群相制（多人当宰相）
  ↓
宋：宰相权力被分割 → 明：废丞相，设内阁 → 清：军机处集权
```

## 🏘️ 地方官职详解

### 一、省级行政（地方最高长官）

| 朝代 | 官职 | 品级 | 管辖范围 | 现代对应 |
|------|------|------|---------|---------|
| **秦** | 郡守 | 二千石 | 郡（约等于省） | 省长 |
| **汉** | 太守 | 二千石 | 郡 | 省长 |
| **汉** | 刺史 | 六百石 | 州（监察区） | 中央巡视组组长 |
| **唐** | 节度使 | 正二品 | 道（大军区） | 大军区司令+省长 |
| **唐** | 观察使 | 正三品 | 道 | 省长 |
| **宋** | 知州 | 正四品 | 州 | 省长 |
| **元** | 行省平章 | 从一品 | 行省 | 省长 |
| **明** | 布政使 | 从二品 | 省 | 省长 |
| **明** | 按察使 | 正三品 | 省 | 省法院院长 |
| **明** | 都指挥使 | 正二品 | 省 | 省军区司令 |
| **清** | 总督 | 正二品→从一品 | 2-3 省 | 大区书记 |
| **清** | 巡抚 | 从二品→正二品 | 1 省 | 省长 |

### 三司分立（明）

```
省级权力三分：
┌─────────────┐
│ 布政使司    │ 管行政、财政（省长）│
│ 按察使司    │ 管司法、监察（省高院）│
│ 都指挥使司  │ 管军事（省军区）│
└─────────────┘
```

### 总督 vs 巡抚（清）

| 对比 | 总督 | 巡抚 |
|------|------|------|
| 管辖 | 2-3 省 | 1 省 |
| 品级 | 正二品（加尚书衔从一品） | 从二品（加侍郎衔正二品） |
| 职责 | 军政、民政 | 民政为主 |
| 现代 | 大区书记（如华东局） | 省长 |
| 别称 | 制台、制军 | 抚台、抚军 |
