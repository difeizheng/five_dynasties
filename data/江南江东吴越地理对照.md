# 🗺️ 古代江南/江东/吴越之地现代对照

*历史地理概念与现代行政区划对照*

---

## 📍 核心概念辨析

### 一、三个概念的关系

```
吴越之地 = 最古老（春秋时期）
    ↓
江东 = 秦汉至隋唐（地理概念）
    ↓
江南 = 唐宋至今（文化+经济概念）
```

| 概念 | 最早出现 | 核心区域 | 现代对应 |
|------|---------|---------|---------|
| **吴越** | 春秋（前 770-前 476） | 苏南/浙江 | 江苏南部 + 浙江 |
| **江东** | 秦汉（前 221-220） | 长江下游南岸 | 苏南/安徽南/浙江/江西 |
| **江南** | 唐宋（618-1279） | 长江以南 | 苏南/浙江/安徽南/江西/上海 |

## 🏛️ 二、江东（秦汉至隋唐）

### 1️⃣ 江东概念

**为什么叫"江东"？**

```
长江在安徽芜湖至江苏南京段呈西南 - 东北流向
从中原视角看，这段长江的"南岸"实际上是"东岸"
所以称为"江东"
```

| 项目 | 内容 |
|------|------|
| **时间** | 秦汉至隋唐（前 221-907） |
| **地理范围** | 长江下游南岸地区 |
| **核心区域** | 苏南、皖南、浙江、赣北 |
| **著名事件** | 项羽起兵、孙权建国 |
