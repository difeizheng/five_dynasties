# 🏙️ 武汉历代政权归属全记录

*从夏朝到 1949 年，武汉（江汉地区）的政权变迁史*

---

## 📍 武汉地理概念说明

**现代武汉** = 武昌 + 汉口 + 汉阳（三镇合一）

**古代对应：**
- **武昌**：历代武昌郡/鄂州治所
- **汉阳**：汉阳府/沔阳州
- **汉口**：明清时期兴起，古代为汉口镇

**核心区域：** 江汉平原、长江与汉水交汇处

### 五代十国武汉归属详解

```

960 年：北宋建立，赵匡胤统一中原
963 年：宋军南下，武汉正式归北宋
```

### 为什么武汉归南唐而不是吴越？

| 对比 | 南唐 | 吴越 |
|------|------|------|
| 地理位置 | 江苏/安徽/江西/湖北 | 浙江/上海 |
| 与武汉距离 | 近（南京到武汉沿江） | 远（杭州到武汉） |
| 军事实力 | 强（十国最强） | 弱（保境安民） |
| 水师 | 强大（长江水师） | 强大（海师） |
| 结果 | ✅ 占据武汉 | ❌ 未触及武汉 |

### 明清"湖广"概念

```
湖广 = 湖北 + 湖南
```

## 🎯 武汉历史地位

### 为什么武汉重要？

| 因素 | 说明 |
|------|------|
| **地理位置** | 长江与汉水交汇，九省通衢 |
| **军事价值** | 控长江中游，南北要冲 |
| **经济地位** | 明清"楚中第一繁盛"，近代工业基地 |
| **政治意义** | 辛亥革命首义之地 |

### 武汉历史关键时刻

```
1. 前 278 年：秦拔郢，武汉入中原版图
2. 208 年：赤壁之战，武汉成三国鼎立前沿
3. 874 年：设武昌军，"武昌"之名始用
4. 963 年：归北宋，结束五代割据
5. 1861 年：汉口开埠，近代化开始
6. 1911 年：武昌起义，推翻帝制
7. 1949 年：武汉解放，新中国时期开始
```

## 💡 看《太平年》与武汉的关系

### 五代十国时期武汉归属

```
《太平年》主要讲吴越国（浙江），但武汉当时属于：

907-924 年：后梁（名义）
924-951 年：楚国（马殷）
951-960 年：南唐（李璟）
960-963 年：北宋

所以《太平年》剧情与武汉无直接关系，
但南唐是吴越的敌人，武汉是南唐西部重镇。
```
